Sobre

Dashboard Covid19

Este Dashboard da Covid19 fornece uma visão geral do novo surto do novo Coronavírus COVID-19 em Cabo Verde. Este dashboard foi construído com R usando a estrutura R Markdown.

Autoria

O dashboard foi desenvolvido e é mantido por Mário Cardoso.

Dados

Os dados de entrada para este dashboard são os pacotes R covid19cvdata e coronavirus R (versão de desenvolvimento). Os dados e o dashboard são atualizados diariamente.

Fonte de dados

Os dados brutos do pacote covid19cvdata são obtidos do website covi19.cv e o pacote coronavírus do Centro de Ciência e Engenharia da Universidade Johns Hopkins (JHU CCSE).

Pacotes

  • Interface do dashboard - o pacote flexdashboard.
  • Visualização - o pacote plotly para os gráficos e mapview para o mapa
  • Manipulação de dados - dplyr e tidyr
  • Tabelas - o pacote DT
