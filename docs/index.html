Sobre

Dashboard Covid19CV

Este Dashboard da Covid19 pretende fornecer uma visão geral do novo surto do novo Coronavírus COVID-19 em Cabo Verde. O dashboard foi construído com R usando a estrutura R Markdown.

Autoria

O dashboard foi desenvolvido e é mantido por Mário Cardoso.

Dados

Os dados de entrada para este dashboard são os pacotes R covid19cvdata e coronavirus (versão de desenvolvimento). Os dados e o dashboard são atualizados diariamente.

Fonte de dados

Os dados brutos do pacote covid19cvdata são obtidos da plataforma pública covi19.cv e os dados do pacote coronavirus do Centro de Ciência e Engenharia da Universidade Johns Hopkins (JHU CCSE).

Pacotes

  • Interface do dashboard - o pacote flexdashboard.
  • Visualização - o pacote plotly para os gráficos e mapview para o mapa
  • Manipulação de dados - dplyr e tidyr
  • Tabelas - o pacote DT
